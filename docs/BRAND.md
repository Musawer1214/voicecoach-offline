# Brand Notes

VoiceCoach Offline uses a quiet desktop-tool style: dark navigation, light work surfaces, teal action states, and compact scorecards.

## Assets

- `assets/brand/voicecoach-logo.svg`
- `assets/brand/voicecoach-readme-hero.svg`
- `build/icon.ico`
- `build/icon.png`

## Visual Direction

- Primary color: `#14b8a6`
- Dark navigation: `#101828`
- Work surface: `#f7f8fb`
- Warning state: `#f59e0b`
- Danger state: `#dc2626`

The UI should stay practical and focused. Use cards for repeated items, scorecards, and review panels; avoid marketing-style landing sections inside the app.
